# Prayer Times App 

A web application that displays daily prayer times based on the selected city using real-time data from an external API.

---

## Project Overview
This project helps users easily view accurate prayer times according to their location.  
The application focuses on clean UI, simplicity, and dynamic data handling.

---
##  Tech Stack
- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- Prayer Times API (Aladhan API)

---

##  Features
- Fetch prayer times dynamically by city
- Display all daily prayer times clearly
- Simple and clean user interface
- Responsive design for all devices
- Error handling for API requests

---

##  Concepts Used
- React Components
- Props & State Management
- useEffect & useState Hooks
- API Fetching
- Conditional Rendering
- Clean UI structure
